# Workflow: /plan

## Purpose
Capture unstructured ideas, concepts, and high-level goals.

## Action
1. Receive the user's unstructured idea.
2. Create a new topic directory in the `Planned` area.
3. Generate a `topic.md` file using the template.
4. Populate the `topic.md` with the user's idea as the "one-liner" description.
5. Confirm the topic has been stored in `Planned`.

## Verification
- Confirm the topic exists in the `Planned` area.
- Confirm `topic.md` contains the description.
