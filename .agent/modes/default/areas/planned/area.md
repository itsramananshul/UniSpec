# Planned

## Purpose
This area serves as the initial container for unstructured ideas, high-level concepts, and feature requests. It is the "brainstorming" phase before any formal specification is written.

## Guidelines
- Capture ideas as they come.
- Use this area to hold discussions with the agent about big concepts.
- Once an idea is ready to be developed, use the `/spec` command to move it to the Staging area.
