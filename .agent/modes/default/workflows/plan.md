# Workflow: /plan

## Purpose
Capture unstructured ideas, concepts, and high-level goals.

## Action
1. Receive the user's unstructured idea.
2. Use `unispec_nav` to confirm the `Planned` area exists.
3. Create a new topic directory in the `Planned` area.
4. Generate a `topic.md` file using the template.
5. Populate the `topic.md` with the user's idea as the "one-liner" description.
6. Confirm the topic has been stored in `Planned`.

## Verification
- Confirm the topic exists in the `Planned` area using `unispec_nav`.
- Confirm `topic.md` contains the description.
